# Notebooks de Experimentación

Espacio para el análisis exploratorio de datos y pruebas rápidas de convergencia.

## Protocolo de uso
- No se debe subir código de producción aquí; una vez validado un método, debe migrarse a `src/`.
- Nomenclatura: `[id]_[descripcion].ipynb` (ej: `01_eda_kepler.ipynb`).
